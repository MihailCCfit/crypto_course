# Лабораторная работа 3 — Шифр Вернама и поточные шифры

Реализация поточных шифров для шифрования файлов.

## Возможности

1. **Генератор ключа** — линейный конгруэнтный генератор (LCG) для создания файла-ключа
2. **Шифр Вернама** — побайтовый XOR двух файлов (зашифрование = расшифрование)
3. **RC4** — реализация поточного шифра RC4

## Запуск

```bash
python main.py
```
